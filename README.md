# ATEEZ甲板后厨 - 个人博客

一个专为分享 ATEEZ 相关小说和内容的个人博客网站。

## ✨ 功能特性

- 📚 文章列表展示
- 📖 文章详情页
- 🎨 美观的航海主题设计
- 📱 响应式设计，支持手机、平板和电脑
- ⚡ Vite + React 高速构建

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 部署到 Vercel

1. 在 GitHub 上创建新仓库
2. 推送代码到 GitHub
3. 访问 [vercel.com](https://vercel.com)
4. 使用 GitHub 账号登录
5. 导入你的仓库
6. 点击 Deploy！

## 📝 添加新文章

编辑 `src/data/articles.ts` 文件，按照现有格式添加新文章即可。

## 🎨 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
